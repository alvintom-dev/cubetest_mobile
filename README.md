# 🧱 Cube Test App

Cube Test App is a mobile application designed to help manage and run structured tests in a simple and organized way.

It focuses on clarity, consistency, and reliability.

---

# 🎯 What This App Does

- Manage test sites
- Organize test sets
- Run and track tests
- Keep test data structured and easy to access

---

# 💡 Key Idea

The app is built in a **very structured way** so that:
- Features stay organized
- Code is easy to maintain
- New features can be added safely
- Everything stays consistent over time

---

# 🧩 App Structure (Simple View)

The app is divided into 4 main parts:

### 📌 Features
These are the main modules of the app:
- Dashboard
- Sites
- Tests
- Test Sets

Each feature works independently.

---

### 🧠 Core System
Handles the main rules of the app:
- Navigation (moving between pages)
- Error handling
- App settings
- Basic utilities

---

### 🔌 Services
Handles external connections:
- APIs
- Local storage
- Notifications
- Third-party tools

---

### 🎨 Shared Components
Reusable UI elements:
- Buttons
- Cards
- Inputs
- Loading indicators

---

# 🔁 How the App Works

When you use the app:

1. You interact with the UI (screen)
2. The app processes your action
3. It fetches or updates data
4. The result is shown back on screen

---

# 🧪 Testing

The app includes automated testing to ensure everything works correctly.

We test:
- Business logic
- App behavior
- Data handling

This helps keep the app stable and reliable.

# 🚀 Technology Used

- Flutter (mobile app framework)

- Clean and structured architecture design

The goal of this app is:
